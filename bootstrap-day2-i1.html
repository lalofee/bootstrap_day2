For basic usage, you can download the compiled and minified CSS, JavaScript, and fonts.Just copy and paste the files you need to your CSS folder. You can work with Bootstrap CDN too. In this case, you just copy thelinks to your header-section.

You can style your page header with a simple class "page-header" and use "h1" tags for your headline. You can utilize the "h1" tag with the attribute "small" for subtext.

The basic grid system in bootraps works with rows and colums. By default there a 12 columns. You can split the content with this grid system as you like. You can also make your content responsive for different media screens.

Navbars are responsive meta components that serve as navigation headers for your application or site.
